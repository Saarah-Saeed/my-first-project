# E-mail Spam Detection Model

A Machine Learning project that classifies e-mails as **Spam** or **Not Spam (Ham)** using Natural Language Processing (NLP) techniques.

## Features

* Detects spam e-mails automatically
* Text preprocessing and cleaning
* Machine Learning based classification
* User-friendly prediction system

## Technologies Used

* Python
* Pandas
* NumPy
* Scikit-learn
* NLTK
* Jupyter Notebook

## Project Workflow

1. Data Collection
2. Data Cleaning
3. Text Preprocessing
4. Feature Extraction
5. Model Training
6. Model Evaluation
7. Spam Prediction

## Dataset

The model is trained on a dataset containing spam and legitimate e-mails.

## Results

The trained model can classify incoming e-mails and predict whether they are spam or not.
